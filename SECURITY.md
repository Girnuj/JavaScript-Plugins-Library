# Política de Seguridad

Gracias por ayudar a mantener este proyecto seguro.

## Reporte de Vulnerabilidades

Si encuentras una vulnerabilidad de seguridad en alguno de los plugins o en el repositorio, por favor sigue estos pasos:

1. **No abras un issue público.**
2. Envía un correo electrónico a: [montenegrosamuel77@gmail.com](mailto:montenegrosamuel77@gmail.com) con los detalles del problema encontrado.
3. Proporciona la mayor cantidad de información posible para poder reproducir y solucionar el problema (incluye versión del plugin, entorno, pasos para reproducir, etc).
4. Espera confirmación de recepción y, si es necesario, colabora en el proceso de solución.

## Buenas Prácticas

- No compartas públicamente detalles de vulnerabilidades hasta que hayan sido corregidas y comunicadas.
- Usa siempre la última versión de los plugins.
- Si tu reporte resulta en una mejora de seguridad, se te reconocerá en el changelog (a menos que prefieras anonimato).

## Alcance

Esta política aplica a todo el código y plugins incluidos en este repositorio.
